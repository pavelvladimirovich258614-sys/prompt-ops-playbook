# Skill: Color Expert

> Claude Code-скилл, который превращает Claude в эксперта по науке о цвете

**Источник:** https://github.com/meodai/skill.color-expert

---

## Что это?

Color Expert — это скилл для Claude Code, который даёт доступ к **113 справочным файлам** и **286 000 словам** о науке о цвете. От работ Гельмгольца (1856) до современного OKLCH.

## Использование

После установки Claude автоматически получает доступ к знаниям о цвете:

```
Пользователь: Какой цвет лучше для кнопки призыва к действию?

Claude (с Color Expert): 
- Для CTA кнопок рекомендуется высококонтрастный цвет
- В OKLCH: L около 60-70%, C максимальный для оттенка
- Красный (#FF4444) — срочность
- Зелёный (#44FF44) — подтверждение
- Оранжевый (#FF8844) — универсальный CTA
```

## Примеры использования

### Пример 1: Выбор цветовой палитры

```
Пользователь: Создай палитру для медицинского приложения

Claude: 
- Основной: #0066CC (доверие, профессионализм)
- Акцент: #00AA44 (здоровье, безопасность)
- Фон: #F8F9FA (чистота, простор)
- Текст: #212529 (читаемость)
- Ошибка: #DC3545 (стандарт)
- Предупреждение: #FFC107 (внимание)
```

### Пример 2: Проверка контраста

```
Пользователь: Проверь контраст #FF5733 на белом фоне

Claude:
- Контраст: 4.5:1
- WCAG AA: ✅ проходит
- WCAG AAA: ❌ не проходит для малого текста
- Рекомендация: использовать для заголовков или увеличить размер
```

### Пример 3: Конвертация цветов

```
Пользователь: Конвертируй #FF5733 в OKLCH

Claude:
- HEX: #FF5733
- RGB: rgb(255, 87, 51)
- HSL: hsl(11, 100%, 60%)
- OKLCH: oklch(66% 0.22 45)
- LAB: lab(60% 55 45)
```
