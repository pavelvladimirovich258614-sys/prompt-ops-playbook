# Prompt – Choose Skill

Используй этот промпт, когда нужно выбрать skill под задачу.

## Промпт

Работай по репозиторию `prompt-ops-playbook`.

Сначала используй:
– `PLAYBOOK.md`
– `INDEX.md`
– `07-registry/skills-and-mcp-registry.md`
– `02-skills/skills-setup.md`

Моя задача:
{описание задачи}

Выбери skill по правилам:
1. Сначала проверь, нужен ли skill вообще.
2. Затем найди 1–3 подходящих варианта в реестре.
3. Выбери тот, который даёт измеримое усиление.
4. Не предлагай skill "на всякий случай".

Ответ дай в формате:
A. Лучший skill
B. Почему он подходит
C. Как подключать пошагово
D. Как проверить на коротком тест-кейсе
E. Когда этот skill не нужен
F. Какие есть альтернативы в реестре
