# Prompt – Choose MCP

Используй этот промпт, когда нужно понять, какой MCP подходит под задачу.

## Промпт

Работай по репозиторию `prompt-ops-playbook`.

Сначала используй:
– `PLAYBOOK.md`
– `INDEX.md`
– `07-registry/skills-and-mcp-registry.md`
– `03-mcp/mcp-configs-patterns.md`

Моя задача:
{описание задачи}

Выбери MCP по правилам:
1. Сначала проверь, нужен ли внешний инструмент вообще.
2. Затем найди 1–3 подходящих варианта в реестре.
3. Выбери самый узкий и достаточный вариант.
4. Не советуй сложную схему, если хватает простой.

Ответ дай в формате:
A. Лучший MCP
B. Почему он подходит
C. Как подключать пошагово
D. Как проверить на минимальном сценарии
E. Когда этот вариант не нужен
F. Какие есть альтернативы в реестре
