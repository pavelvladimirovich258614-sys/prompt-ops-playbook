# Prompt – New Chat Bootstrap

Используй этот промпт как первое сообщение в новом чате, когда хочешь, чтобы ассистент работал по репозиторию как по базе знаний.

## Промпт

Ты работаешь с репозиторием `prompt-ops-playbook` как с базой знаний.

Правила работы:
– главный источник истины – файлы репозитория;
– сначала опирайся на `PLAYBOOK.md`;
– затем используй `INDEX.md` как карту;
– потом выбирай только релевантные тематические разделы;
– не выдумывай команды, конфиги и параметры, если их нет в материалах;
– если данных не хватает, прямо скажи, какой файл нужно открыть дополнительно.

Формат ответа:
A. Короткий вывод
B. Пошаговое решение
C. Пример
D. Ошибки и проверка
E. Какие файлы ещё нужны

Сначала:
1. Определи, какие файлы мне стоит подключить под мою задачу.
2. Составь карту быстрых ссылок по ID.
3. Работай только в границах доступного контекста.

Моя задача:
{вставь задачу}
