# 08 – Prompts

Этот раздел содержит готовые стартовые промпты для работы с базой знаний.

Его задача – ускорить запуск новых чатов и сделать поведение ассистента повторяемым.

Промпты в этом разделе нужны не для "магии", а для правильного подключения контекста:
– какой файл открыть;
– как сформулировать задачу;
– как заставить ассистента сначала смотреть в репозиторий;
– как получать прикладной ответ, а не абстрактные советы.

## Какие промпты хранятся в разделе

– `new-chat-bootstrap.md` – старт нового чата
– `choose-mcp.md` – выбор MCP под задачу
– `choose-skill.md` – выбор skill под задачу
– `subagents-design.md` – разбиение на субагентов
– `diagnostics.md` – диагностика проблемы
– `repo-ingestion.md` – как просить ассистента работать по репозиторию как по источнику истины

## Принцип использования
Не нужно вставлять все промпты сразу.
Берётся один промпт под конкретную задачу и при необходимости дополняется нужными файлами из репозитория.
