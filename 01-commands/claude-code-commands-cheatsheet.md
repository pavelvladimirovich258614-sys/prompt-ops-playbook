# 50+ команд Claude Code — Шпаргалка

> Полный справочник команд Claude Code для эффективной работы

## 🚀 Project Setup & Memory (Настройка проекта)

| Команда | Описание |
|---------|----------|
| `/init` | Создаёт CLAUDE.md с описанием проекта, стека и правил кодинга |
| `/memory` | Открыть CLAUDE.md для редактирования |
| `/add-dir` | Добавить рабочие директории в сессию |
| `#text` | Добавить строку в CLAUDE.md без открытия редактора |

**Пример:**
```
/init              # Создать CLAUDE.md для проекта
/memory            # Отредактировать память проекта
/add-dir ./src     # Добавить папку src в сессию
#text Используем TypeScript strict mode
```

## 🔄 Context & Session Lifecycle (Управление сессией)

| Команда | Описание |
|---------|----------|
| `/compact` | Сжимает историю диалога, освобождает контекст |
| `/clear` | Очищает всю историю |
| `/context` | Показывает использование контекста (%) |
| `/resume` | Продолжить предыдущую сессию |
| `/rewind` | Откатить диалог/код |
| `/fork` | Создать ветку диалога |
| `/rename` | Авто-имя сессии |
| `/color` | Задать цвет prompt-бара |

**Пример:**
```
/compact           # Сжать контекст когда заполнен на 80%+
/clear             # Начать с чистого листа
/context           # Проверить сколько контекста использовано
/rewind 5          # Откатить последние 5 сообщений
```

## 💻 CLI Flags (запуск из терминала)

| Флаг | Описание |
|------|----------|
| `claude` | Интерактивная сессия |
| `claude "prompt"` | Запуск с промптом |
| `--print` / `-p` | Выполнить и выйти |
| `-c` / `--continue` | Продолжить последнюю сессию |
| `--resume` | Восстановить сессию |
| `--from-pr` | Сессия из PR |
| `--append-system-prompt` | Добавить инструкции |
| `--system-prompt` | Полностью переопределить инструкции |
| `--dangerously-skip-permissions` | Пропустить проверки |
| `--agents` | Задать саб-агентов (JSON) |
| `--output-format json` | Вывод в JSON |
| `--effort` | Уровень размышления (low → max) |
| `--worktree` | Изолированный git worktree |
| `--remote-control` | Управление через web |
| `--debug` | Логирование |
| `--version` | Версия |

**Пример:**
```bash
claude                              # Интерактивный режим
claude "Создай React компонент"     # Одноразовый запуск
claude -p "Fix bug"                 # Выполнить и выйти
claude --continue                   # Продолжить сессию
claude --effort max "Сложная задача" # Максимальные рассуждения
```
