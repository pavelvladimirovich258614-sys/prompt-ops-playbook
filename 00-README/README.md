# 00 – README

Этот раздел содержит вводные пояснения о том, как устроена база знаний, зачем она нужна и как с ней работать в ежедневной практике.

Основная идея проста: большой курс в одном документе неудобен для ИИ, потому что он тяжело переносится между чатами, плохо масштабируется и мешает быстро находить нужные фрагменты. Поэтому курс переводится в модульную структуру, где каждая тема лежит отдельно и может использоваться независимо.

Задача этого раздела – объяснить принципы организации всей системы знаний и зафиксировать, почему репозиторий устроен именно так.
